Data Profiling using Intelligent Cloud Services

I began by logging into Informatica Intelligent Cloud Services.

After accessing the platform, I selected the Data Profiling option.

Step 1: Connect to Oracle Database

To facilitate the process, I used SQL Developer to connect to the Oracle database, which I downloaded and installed from Oracle's official site.

I obtained the necessary credentials and connection details from Oracle SQL Developer.

Step 2: Install Informatica Cloud Secure Agent

Next, I installed and ran the Informatica Cloud Secure Agent, which enabled me to establish a connection to the Oracle Database.

Step 3: Data Profiling

Once the connection was set up, I ran a profile using the Data Profiling tool to gather insights into the quality and structure of the data.

Step 4: Reviewing the Results

The results were displayed clearly, offering details about the data.

Step 5: Comparing Data Quality

I was able to gain several insights from this analysis by comparing the columns' qualities.

Step 6: Profile Comparison

Additionally, I compared profiles in terms of their quality.
