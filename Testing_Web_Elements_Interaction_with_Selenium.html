Testing Web Elements Interaction with Selenium

I used the console to look at the HTML code and find elements I wanted to interact with, such as the search bar.

I started by setting up a simple Selenium test environment. First, I imported the necessary libraries for the test, including unittest, By, Keys, and a custom library google_colab_selenium.

I then created a test class called GoogleSearchTest and defined a setup method where I initialized the Chrome driver using gs.UndetectedChrome().

Next, I created the test_search_selenium method to automate the process of searching for the term "Selenium" on Google. I opened Google's homepage by calling self.driver.get("https://www.google.com/"). Then, I accessed the search bar element by its name attribute using self.driver.find_element(By.NAME, "q").

Once I had access to the search bar, I sent the search term "Selenium" followed by pressing the Enter key, using send_keys("Selenium" + Keys.ENTER). After that, I added a brief pause using time.sleep(2) to give the search results time to load.

To verify that the search results were correct, I accessed the first result using a CSS selector, self.driver.find_element(By.CSS_SELECTOR, "h3"). I printed the text of the first result to the console and checked if it contained the word "Selenium" with an assertion: self.assertIn("Selenium", first_result.text, "Selenium not found in the first result").

Finally, I included a tearDown method to close the browser after the test with self.driver.quit(). To run the tests, I created a run_tests function and used unittest.TextTestRunner() to execute the test case.


import unittest
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys
import google_colab_selenium as gs
import time

class GoogleSearchTest(unittest.TestCase):

    def setUp(self):
        """Set up the test environment."""
        # Initialize the driver
        self.driver = gs.UndetectedChrome()

    def test_search_selenium(self):
        """Test that searching for 'Selenium' on Google works."""
        # Open the website
        self.driver.get("https://www.google.com/")

        # Find the search bar element
        search_bar = self.driver.find_element(By.NAME, "q")

        # Perform the search by entering "Selenium"
        search_bar.send_keys("Selenium" + Keys.ENTER)

        # Wait for the results to load (can be adjusted based on the network speed)
        time.sleep(2)

        # Verify that results are returned by checking if the first result contains the word 'Selenium'
        first_result = self.driver.find_element(By.CSS_SELECTOR, "h3")
        print(f"First search result text: {first_result.text}")
        self.assertIn("Selenium", first_result.text, "Selenium not found in the first result")

    def tearDown(self):
        """Clean up after the test."""
        self.driver.quit()

# Run the tests
def run_tests():
    # This prevents the error that occurs in Jupyter environments
    unittest.TextTestRunner().run(unittest.TestLoader().loadTestsFromTestCase(GoogleSearchTest))

run_tests()
