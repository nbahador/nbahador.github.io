Creating stored procedures in Oracle SQL Developer

In this simple demo, I tried to answer the question: "How can we transform a given string into uppercase?" The goal was to ensure standardization and consistency in user input, especially when dealing with data that requires a uniform format for processing or display.

To answer this question, I needed information on how to perform string manipulation using SQL procedures, particularly in Oracle SQL. Specifically, I needed to understand how to create a stored procedure that accepts an input string, converts it to uppercase, and outputs the result.

The query that helped me retrieve the necessary information was the following:

CREATE OR REPLACE PROCEDURE P_TO_UPPER (P_STR VARCHAR2)
IS
    L_STR VARCHAR2(50);
BEGIN
    L_STR := UPPER(P_STR); -- Convert input string to uppercase
    DBMS_OUTPUT.PUT_LINE('Input string in Upper case: ' || L_STR); -- Display the result
END P_TO_UPPER;

I used this query to create the stored procedure P_TO_UPPER, which accepts a string input, converts it to uppercase using the UPPER() function, and outputs the result via DBMS_OUTPUT.PUT_LINE. After creating the procedure, I executed it with an input string (e.g., 'hello world'), which returned the desired uppercase version ('HELLO WORLD').
